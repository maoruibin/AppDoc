# 云备份

点亮 是一个本地习惯打卡应用，数据均存储在手机本地，为了保证数据安全、点亮 做了很多工作，详情可查看[数据备份](data_backup.md)，但数据只要存储在本地，就有丢失的可能，比如手机坏了，死机了或者丢了。

针对上面的问题，**点亮 支持了个人云盘**，只要在 点亮 中配置好个人云盘，就可以将 点亮 数据同步到个人云端。

目前 点亮 支持的云盘是 WebDAV 云盘。

这里推荐你使用国内的坚果云，免费额度就可以完全满足 点亮 的数据同步场景，具体如何操作，可查看教程：[如何配置 WebDAV 云盘](./lesson-webdav.md)

## 相关问题

### 如何把所有数据打包转移到另一个手机？
查看 [如何备份还原本地数据](./data_recovery.md)

### 备份、推送，具体是什么操作？
备份时，会生成一份所有习惯、打卡记录等的数据，保存到一个本地文件，作为本地备份。

如果配置了个人云端，会把本地备份文件推送至云端。如果没有配置个人云盘，这里只会进行本地备份。

### 备份的具体数据有哪些？
1. 习惯数据
2. 打卡记录数据
3. 积分和愿望数据
4. 本地的偏好配置
5. 其他数据，如习惯归档关联等
