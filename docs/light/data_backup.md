#  数据备份

点亮 是一个本地习惯打卡应用，所有数据默认均存储在本地数据库，该数据库存储在 Android 文件沙盒，只有 点亮 可以访问，其他 APP 均访问不到（Root手机除外）。

## 本地备份

点亮 会自动打包所有的习惯打卡数据到一个文件中，并对数据进行加密，然后存储在本地文件夹下，目录为：
> /storage/emulated/0/Documents/LightPlus/Backups

因为默认数据都存储在 APP 的文件沙盒中，该文件会随着 APP 卸载被删除，所以这里通过该机制，将数据外置备份至外部磁盘，防止打卡数据丢失，同时为了防止明文数据泄露，这里对数据进行了 AES 加密。

当下次重新安装 点亮 时，点亮 会自动检查备份文件夹，并提示用户进行数据恢复。


## 云备份

上述措施已经尽可能的做到了数据安全，但是数据只要在手机上，就还有可能丢失，比如手机突然坏了，丢了，这时数据是无法找回的，如果数据此时在云端，就可以避免这个问题。点亮 没有自己的云端数据中心，为了解决这个问题，点亮 使用了行业内比较成熟的方案：WebDAV。
