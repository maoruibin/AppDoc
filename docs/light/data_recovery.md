# 如何备份还原本地数据

## 自动备份

每次打卡操作都会在本地文件夹中自动创建一份备份文件，位置为：
> /storage/emulated/0/Documents/LightPlus/Backups

## 手动备份

在设置页面，点击「数据管理」→「手动备份」，可以立即生成一份备份。

## 恢复数据

### 从本地备份恢复

1. 安装 点亮 后，打开 App
2. 如果检测到本地有备份文件，会自动提示恢复
3. 也可以在设置页面，点击「数据管理」→「从本地恢复」

### 从云端备份恢复

1. 在设置中配置好 WebDAV 云盘账号
2. 点击「从云端恢复」
3. 选择要恢复的备份文件即可
