# 如何在网页端、客户端同步数据
inBox 支持两种同步模式：

- **自动同步（auto）**：App 启动时自动同步 + 数据变更时自动同步，推荐日常使用
- **手动同步（manual）**：仅在用户手动触发时同步

> 可在 **数据云中心** 页面中切换同步模式。

如果数据量小，建议**开启自动同步**，这样每次数据变化时都会第一时间同步到云端，打开 App 也会自动拉取最新数据。

<img src="../public/img/dataoperator1.png" width="80%" alt="">

如果数据比较多，可以切换到手动同步，需要时手动拉取和推送数据。

## 手动控制

在手机端写笔记时，首先拉取一次最新的数据，然后记笔记，记完后，再手动推送数据，这样就可以让远端数据保持最新。如果平时不用网页版，则不需要这样操作

<img src="../public/img/dataoperator2.png" width="80%" alt="">

当在网页端发布完笔记，想要在手机上查看，那么就先手动拉取一次数据，既可获取到最新的网页端的改动。
